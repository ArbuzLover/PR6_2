## 🔧 Рефакторинг кода

Исходный код содержал логику вычислений непосредственно в обработчиках событий страниц WPF. Для обеспечения возможности модульного тестирования была выполнена реорганизация кода.

### Созданные классы-калькуляторы

| Класс | Назначение |
|-------|------------|
| `Page1Calculator` | Вычисления для первой страницы |
| `Page2Calculator` | Вычисления для второй страницы |
| `Page3Calculator` | Вычисления для третьей страницы |

### Тесты для Page1 (5 тестов)

| № | Название теста | Тип | Проверяемый сценарий | Ожидаемый результат |
|---|----------------|-----|----------------------|---------------------|
| 1 | `Calculate_ValidInputs_ReturnsTrue` | ✅ Позитивный | Корректные входные данные | true |
| 2 | `Calculate_ValidInputs_ResultMatchesExpected` | ✅ Позитивный | Точность вычислений | Результат с погрешностью 0.0001 |
| 3 | `Calculate_NegativeFraction_HandlesCorrectly` | ✅ Позитивный | Отрицательная дробь (кубический корень) | true, результат > 0 |
| 4 | `Calculate_SinZero_ReturnsFalse` | ❌ Негативный | Деление на ноль (sin(z)=0) | false |


### Тесты для Page2 (4 теста)

| № | Название теста | Тип | Проверяемый сценарий | Ожидаемый результат |
|---|----------------|-----|----------------------|---------------------|
| 1 | `Calculate_XqGreaterThan10_ReturnsLog` | ✅ Позитивный | Ветка |x·m| > 10 (логарифм) | true, ln(|fx|+|m|) |
| 2 | `Calculate_XqLessThan10_ReturnsExp` | ✅ Позитивный | Ветка |x·m| < 10 (экспонента) | true, e^(fx+m) |
| 3 | `Calculate_XqEquals10_ReturnsSum` | ✅ Позитивный | Ветка |x·m| ≈ 10 (сумма) | true, fx+m |
| 4 | `Calculate_LogArgumentZero_ReturnsFalse` | ❌ Негативный | Аргумент логарифма ≤ 0 | false |
